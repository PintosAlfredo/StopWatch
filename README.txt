# JavaScript Stopwatch

This project is a simple stopwatch built using HTML, CSS, and JavaScript. It allows users to start, stop, and reset a timer, displaying the elapsed time in a user-friendly format.

## Features

- **Start/Stop Functionality**: Users can start and stop the timer with a single button.
- **Reset Functionality**: A separate button allows users to reset the timer to zero.
- **Formatted Time Display**: The stopwatch displays time in `HH:MM:SS` format.
